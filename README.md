D3.js realtime chart demo
=========================

Installation
------------

Optionally,

```
npm install -g electron-prebuilt
```

Then,

```
npm install
```


Usage
-----

```
electron .
```


Issues
------

Chart seems not to erase quickly enough, creating a "doubling" artifact. This may just be a POV issue though...
